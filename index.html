Amazing "Star Wars" Facts

Yoda's original name was Buffy

In the very, very early writing stages of The Empire Strikes Back, Yoda’s original name was—brace yourself—Buffy. Yes, Buffy, a name we usually associate with awesome teenage vampire slayers. The world would have been a very different place if Yoda had somehow been given this very un-Yoda-like name.
